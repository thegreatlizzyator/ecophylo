Having Deme abundance¶

ecophylo.getAbund(tree, samples=None)¶
Parameters

  • tree ((ete3 class)) – Phylogeny with attributes on leafs. This attributes is a character string containing all names of the species individual (mean to use topPhylo result). Names is formated like this : ” name1 name2 name3”

  • samples (int) – number of individual in the community # TODO : set this check as optionnal

Returns

sfs – DESCRIPTION.

Return type

TYPE

Examples

>>> from ete3 import Tree
>>> tree = Tree('(((A:5,(B:3, C:3))1:2,(D:2, E:2)1:5)1:2, (F:3, G:3)1:6);')
>>> import ecophylo as eco
>>> phylo = eco.toPhylo(tree, 0.5, seed = 42)
>>> print(phylo)

      /-sp1
   /-|
--|   \-sp2
  |
   \-sp3
>>> getAbund(phylo, 7)
[3, 2, 2]

Having species diversity¶

ecophylo.getDeme(tree, div=False)¶
Parameters

  • tree ((ete3 class)) – Phylogeny with attributes on leafs. This attributes is a character string containing all names of the species individual (mean to use topPhylo result). Names is formated like this : ” name1 name2 name3”

  • div (bool) – Option to simplify the matrix to a simple list of Deme species diversity.

Returns

indiv – site/species matrix of the tree

Return type

nest list of int

Examples

>>> from ete3 import Tree
>>> tree = Tree('(((A_0:5,(B_0:3, C_1:3))1:2,(D_1:2, E_1:2)1:5)1:2, (F_2:3, G_2:3)1:6);')
>>> import ecophylo as eco
>>> phylo = eco.toPhylo(tree, 0.5, seed = 42)
>>> print(phylo)

      /-sp1
   /-|
--|   \-sp2
  |
   \-sp3
>>> getDeme(phylo)
[[2, 1, 0], [0, 2, 0], [0, 0, 2]]
>>> getDeme(phylo, div = True)
[1, 2, 1]
